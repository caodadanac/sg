Second Task: Creating the Snake

The snake works like the fruit. Except that we store and draw multiple vectors.

  • First, create a class named Snake. It must contain a constructor. Inside the constructor create 3 consecutive vectors. Ex: Vector2(5,10),Vector2(4,10),Vector2(3,10)
  • Next, create a function named draw_snake. Inside the function create a for loop to create each block of snake, then draw the rectangle.

Third Task: Moving the Snake

Similar to the algorithm for moving an enemy to the left, which means when the enemy goes off the screen, we will reset the enemy to the right side. Similarly, for our snake consisting of 3 blocks, to move the snake to the right, we can use the following algorithm:

  • The head is moved to a new block.
  • The block before the head gets the position where the head used to be.
  • Each block is moved to the positon of the block that used to be before it. (deletes the last block)

To achieve that, you can follow these steps:

  • Create a variable called "direction" to represent the movement direction. For example, for moving to the right, you can set the direction vector as (1, 0), indicating movement along the x-axis.
  • Copy the entire body of the snake, except for the last block.
  • Add a new block to the beginning of the snake's body list with the value equal to "snake.body[0] + direction".
